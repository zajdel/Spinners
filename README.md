# Spinners
Code for processing tethered E. coli flagellar rotation and training ML classifiers. Generated data presented at 2018 IEEE Engineering in Medicine and Biology Conference.

Citation: TJ Zajdel, A Nam, J Yuan, VR Shirsat, B Rad, MM Maharbiz, "Applying machine learning to the flagellar motor for biosensing." *2018 40th Annual International Conference of the IEEE Engineering in Medicine and Biology Society (EMBC)*, 2018.

https://ieeexplore.ieee.org/abstract/document/8512907

Code overview in Appendix B of "Electronic Interfaces for Bacteria-Based Biosensing" dissertation thesis by TJ Zajdel. Uploaded to repo as spinner_processing_appendix.pdf.
